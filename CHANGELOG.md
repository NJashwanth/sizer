# Changelog

## [0.0.1] - 2026-06-16

### Added

- File size badges in the VS Code explorer
- Smart formatting: B, KB, MB, GB with 1 decimal place
- Bytes-only format option
- Real-time auto-refresh via FileSystemWatcher
- User-configurable settings (showSize, sizeFormat)
- Manual refresh command via Command Palette

### Future Ideas

- Color-code badges by file size ranges
- Configurable exclude patterns (e.g., skip node_modules)
- Size threshold filtering (only show badges above N bytes)
- Status bar widget showing total workspace size
- Custom tree view sorted by file size
- Per-language size statistics
